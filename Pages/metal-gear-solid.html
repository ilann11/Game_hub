Metal Gear Solid

1998 Konami 4.9/5

Description

Metal Gear Solid est un jeu d'infiltration révolutionnaire qui a redéfini le genre. Les joueurs incarnent Solid Snake, un agent spécial infiltré dans une base militaire pour empêcher une menace nucléaire. Avec son scénario complexe, ses personnages charismatiques et son gameplay innovant basé sur la furtivité, le jeu est devenu une référence incontournable.

Caractéristiques

  • Gameplay furtif innovant
  • Histoire cinématographique
  • Personnages mémorables
  • Combats de boss uniques
  • Bande-son légendaire
